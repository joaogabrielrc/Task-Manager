## Setup Instructions

### Run project with Docker Compose

```bash
docker compose up -d --build
```

Or

### Install [Node.js](https://nodejs.org/en/) modules

```bash
npm install
```
### Start the web server

To start the web server you need to run the following command.

```bash
npm run dev

```
